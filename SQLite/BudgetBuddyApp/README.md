https://github.com/betomoedano/Budget-Buddy-App

## Setup DB
```bash
sqlite3 -version

sqlite3 mySQLiteDB.db

npx expo install expo-sqlite expo-file-system expo-asset

```
put mySQLiteDB.db under assets folder.
